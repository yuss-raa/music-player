## 📦 Installation
 
### What You'll Need
 
- **Node.js**
- **npm**
 
### Setup Steps
 
1. **Clone this repo**
```bash
git clone https://github.com/yusraSyed05/music-player.git
```
 
2. **Go to the project folder**
```bash
cd music-player
```
 
3. **Install packages**
```bash
npm install
```
 
4. **Add your YouTube API key**
 
Create a `.env` file in the project root:
```
VITE_YT_API_KEY= add api key here 
```
 
5. **Run the dev server**
```bash
npm run dev
```
